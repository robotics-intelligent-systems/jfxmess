# Multi-Energy System Simulation

Data models related to the Energy Domain. OSeMOSYS is a full-fledged systems optimization model for long-run energy planning. OpenFAST is a wind turbine simulation tool. H2Integrate is a Python package for modeling and designing hybrid energy systems. Planning/Design tool for micro grids with hydrogen storage systems. OpenOA is a software framework written in Python for assessing wind plant performance. The WindPowerPlants library is an open source library for the simulation of wind power plants. ThermoSysPro is a library providing model components in the disciplines of thermal hydraulics and instrumentation and control. Modelica library for the simulation of photo voltaic cells and modules. A Modelica library for photovoltaic system and power converter design. A package for Modelica simulating Molten Salt Reactors. The ThermoCycle library is a library for dynamic modelling of thermal systems. preCICE is an open-source coupling library for partitioned multi-physics simulations. PowerGrids, a Modelica library for electro-mechanical modelling of power systems. DSNP (Dynamic Simulator for Nuclear Power-Plants) is a system of programs and data files used to simulate nuclear power plants or parts thereof. DDNRS-ERSN is an Open Nuclear Reactor Simulator and Reactor Core Analysis Tool based on the Version5 of the reactor physics codes DRAGON and DONJON. ClaRa is a Modelica library for simulation of Clausius-Rankine (ClaRa) cycles in power plants. ModelcaGridData generate power grid dynamic simulation data automatically for machine learning applications using Python and Modelica models. MOOSE is an object-oriented C++ finite element framework for developing tightly coupled multiphysics solvers. Armi is an open-source nuclear reactor analysis automation framework that helps design teams increase efficiency and quality. NeqSim is a library for calculation of fluid behavior, phase equilibrium and process simulation

The directory structure includes a specific folder for each phase of development and engineering programs:

- __MBSE__: The root folder Arcadia (ARChitecture Analysis and Design Integrated Approach) is a method of tools dedicated to systems engineering and architecture, supported by the Capella modeling tool.    
  - __CAD__: For computer-aided design.
  - __CAM__: Dedicated to the part manufacturing and assembly process.
  - __CAS__: For simulation of end-to-end functionality and performance analysis.
